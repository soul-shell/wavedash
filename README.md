# wavedash

## Environment

1. Install `inotify-tools` with your system package manager, e.g. `dnf install inotify-tools`.
2. Run `npm i`.

## Development

Open in separate tabs:
* `npm run server`
* `npm run css-watch`
* `npm start`

Go to [localhost:8000](http://localhost:8000/).

## Deployment

1. `npm run pack`
2. `npm run deploy`
